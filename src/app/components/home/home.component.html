Aplicación web(MoTecnology)

Esta es una aplicación web de la cual su función principal es llevar el control de las simcards vendidas por la empresa.
entre sus características se encuentran:
- Administración de usuarios(editar,eliminar,actualizar,mostrar y activar) en la plataforma web.
- Diferentes vistas y funciones de los usuarios dependiendo del role.
- Creación del inventario de las simcards (CRUD).
- creación de los clientes (CRUD)
- Administración y creación de las ventas(CRUD).
- Exportar excel, poder imprimir y exportar a pdf.
- Reportes personalizados(Graficas, Clientes que mas compraro, etc)
- Cargar las simcards correspondientes a cada usuario.
Entre otras mas características.
El lenguaje que se utilizo para el backend fue PHP Con la arquitectura MVC. y para el frontend javascript, jquery.
El enlace de la aplicacion web es http://simcloud.com.co/app/.
